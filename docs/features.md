# 功能清单

## 当前状态（as-is）

- 核心对话体验：发送消息、流式输出、消息渲染、代码高亮复制。
- 核心会话管理：侧边栏会话列表、新建、切换、重命名、删除。
- Claude 风格布局：左侧会话栏 + 右侧聊天区，聚焦高频聊天路径。
- 基础体验增强：亮暗主题切换、移动端适配、侧边栏折叠。
- 历史持久化：会话元信息与消息本地存储，刷新后可恢复。
- 当前仅保留模型流式聊天接口（`/api/chat`）；会话管理前端本地化。

## 对外展示功能

### 1. 流式对话

输入问题后立即触发流式响应，保证连续反馈与低等待感。

### 2. 会话管理

侧边栏提供新建、切换、重命名、删除，覆盖对话整理的核心需求。

### 3. Markdown 与代码阅读

助手消息支持 Markdown、代码高亮与复制，方便技术类内容消费。

### 4. 主题与布局体验

支持亮暗主题、侧边栏折叠与移动端适配，保持跨端一致体验。

## 已下线能力（本次重构移除）

- Agent 面板与运行链路（含检索/草稿/校验步骤）。
- 知识库与 Embedding 检索。
- 会话内搜索高亮与会话洞察统计。
- 标签体系、批量操作、导入/导出。
- Provider 高级设置弹窗。

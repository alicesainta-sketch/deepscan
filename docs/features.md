# 功能清单

## 当前状态（as-is）

- 核心对话体验：发送消息、流式输出、停止生成、重新生成。
- 消息阅读能力：Markdown 渲染、代码高亮、消息/代码复制。
- 核心会话管理：新建、切换、重命名、删除。
- Claude 风格布局：左侧会话栏 + 右侧聊天区，聚焦高频对话路径。
- 基础体验增强：亮暗主题切换、移动端适配、侧边栏折叠。
- 本地持久化：会话元信息与消息分片存储，刷新后可恢复。
- 服务端模型代理：统一通过 `/api/chat` 请求上游模型。

## 暂不包含能力

- 账号体系与多端同步。
- 服务端数据库持久化。
- Agent 工作流（任务编排、工具调用、状态跟踪）。
- MCP（Model Context Protocol）工具接入。

## 开发计划（下一阶段）

- 前后端分离：从当前 Next.js Node API 路由迁移到独立 Go Gin 服务。
- 服务端持久化：引入数据库，支持会话与消息服务端存储。
- Agent：提供可配置的 Agent 执行链路（计划、执行、重试、观测）。
- MCP：接入标准 MCP 协议，统一扩展外部工具能力。

更多排期与里程碑见 `docs/roadmap.md`。
